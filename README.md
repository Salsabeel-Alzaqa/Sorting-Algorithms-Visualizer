# Sorting Algorithms Visualizer:
A web page showing how sorting algorithms work.

Algorithms covered so far:
1) Bubble sort.
2) Merge sort.
3) Selection sort.
4) Insertion sort.
5) Heap sort. 
6) Quick sort.

## Live Demo
- [Sorting Algorithms Visualizer](https://salsabeel-alzaqa.github.io/Sorting-Algorithms-Visualizer/)
